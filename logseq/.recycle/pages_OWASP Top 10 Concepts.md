# OWASP Top 10 2025 - Security Concepts
- ## About OWASP Top 10 2025
  
  The OWASP Top 10 is a standard awareness document for developers and web application security. It represents a broad consensus about the most critical security risks to web applications.
  
  **Key Changes in 2025:**
- SSRF has been rolled into A01 (Broken Access Control)
- Categories have been reordered based on current threat landscape
- New focus on supply chain and data integrity failures
